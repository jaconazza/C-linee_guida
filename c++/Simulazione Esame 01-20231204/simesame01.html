Simulazione di Esame 01

Descrizione del Problema

Vogliamo implementare un programma che gestisce un insieme di record. Ogni record è composto da un nome e da un valore intero. Il programma deve permettere di inserire un nuovo record, di cancellare un record esistente, di modificare il valore di un record esistente, di caricare i records da un file di testo e di salvare i records su un file di testo. Si può assumere che i nomi dei record siano stringhe di lunghezza massima 127.

Il formato dei files di testo è il seguente: ogni riga contiene un record e consiste di due parti separate da uno spazio. La prima parte è il nome del record, la seconda parte è il valore intero. Per esempio, la riga Pippo 5 contiene il record con nome Pippo e valore 5. La prima riga del file contiene il numero di records contenuti nel file.

La funziona di ricerca deve restituire il valore del record con nome dato o -1 se il record non esiste.

Il problema va risolto implementando una serie di funzioni che operano su una struttura dati che rappresenta l'insieme dei records. La struttura dati deve essere implementata come un array dinamico. È consigliato usare un array dinamico che permette l'inserimento in coda in modo efficiente.

Funzioni e Punteggio

  • RecordSet init(): 2 punti
  • crea un array vuoto di records
  • void drop(RecordSet& rs): 1 punti
  • dealloca l'array di records
  • int insert(RecordSet& rs, const char* name, int value): 5 punti
  • inserisce un nuovo record alla fine dell'array solo se il nome non esiste
  • ritorna la posizione del record inserito o -1 se il record esiste già
  • int remove(RecordSet& rs, const char* name): 5 punti
  • rimuove il record con nome dato
  • ritora la posizione del record cambiato o -1 se il record non esiste
  • int update(RecordSet& rs, const char* name, int value): 5 punti
  • modifica il valore del record con nome dato
  • ritora la posizione del record cambiato o -1 se il record non esiste
  • int search(RecordSet& rs, const char* name): 5 punti
  • restituisce il valore del record con nome dato o -1 se il record non esiste
  • int load(RecordSet& rs, const char* filename): 5 punti
  • carica i records dal file di testo
  • ritorna il numero di records caricati
  • int save(RecordSet& rs, const char* filename): 5 punti
  • salva i records sul file di testo
  • ritorna il numero di records salvati

Regole e Test

Il programma deve essere scritto nel file simesame01.cpp. Potete madoficare il main() di quel file e usarlo come volete per testare le funzioni che implementate. Il programma compila e viene lanciato con il comando:

g++ simesame01.cpp -o simesame01 && ./simesame01

Il programma viene testato automaticamente con il comando:

g++ test-simesame01.cpp -o test-simesame01 && ./test-simesame01

Il programma di test collauda le funzioni che avete implementato e assegna un voto a seconda di quanti test passano. Il voto è la somma dei punteggi elencati sopra e dipende solo dalla correttza con cui si passano i test. Si può consegnare solo se il tester ritorna almeno 18. Il voto massimo è 33 che corrisponde a 30 e lode.

Il test deve compilare e deve completare la sua esecuzione. In ogni altra evenienza il voto è 0.
